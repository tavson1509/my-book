adicionando estilos nas tags

na pagina anterior vimos que é possivel adicionar cores aos paragrafos, aos titulos e até mesmo na pagina em si, e se eu te falar que não é nescessário adicionar-los um por um ?
se eu te disse que você poderia por exemplo... fazer com que todos os itulos <h1> fossem de uma cor especifica ?
e sim isso é possivel vamos supor que voce precise q todos texto de todos seus titulos <h1> de sua pagina sejam de uma cor expecifica é simples basta adicionar na part "head" de seu site a tag <style> entre a tag style adicione:
h1{ aqui voce vai digitar: color:escolha uma cor;}
pronto, mas voce pode também adicionar varias outras coisas como o comando background-color:escolha uma cor; assim como qualquer outra coisa sobre estilos em CSS

adiocionando ID nas tags

voce aprendeu a adionar estilos em todas as tags de um determinado valor, mas vomos supor que voce tenha 8 itulos h2 no seu site e voce precise deixar quatro em um estilos dois em outro e mais dois em outro como que faria ?
temos a solução para você, podemos usar a tag
<h2 id="qualquer nome">
pronto sua tag de titulo agora tem um indentificador e sempre que voce quiser modificar todos os titulos com esse identificador basta voce fazer as mesmas coisas do nosso tutorial a cima de adionar
h2#nome-definido {digite aqui os os comandos que deseja usar}

adicionando CLASS

aprendendo adicionar o id, mas o id é limitado apenas para uma unica vez mas e se precisar-mos adicionar varias "classes" do mesmo jeito ?
basta usar a CLASS é praticamente a mesma coisa do ID mas a CLASS voce pode adicionar quantas vezes voce quiser basta voce usar a esma tag a cima usada, exemplo
<h2 class="qualque nome"> e para voce usar é do mesmo jeito que usou no exemplo a cima...

escondendo textos

se voce deseja ocultar um texto até que o usuario passe a mouse por cima tipo um menu de opções voce pode é simples, voce ira adicionar um titulo e nesse titulo um paragrafo na area de style do seu codigo CSS voce ira usar o comando h1>p {tags desejadas}
o comando h1>p serve para voce dizer que voce deseja fazer com os <p> dentro de <h1>...
mas o que é preciso fazer para esconder os <p> dentro do meu titulo e só aparecer quando eu passar o mouse por cima ?
muito simples primeiro vamos esconder basta usar a tag
display: blok;
e para somente aparecer ao passar o mouse por cima dentro da area de <style> basta adicionar
h1:hover>p {display: blok;}
