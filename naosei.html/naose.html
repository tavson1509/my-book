testando cargas de imagens e videos

carregando imagens internas

abaixo voce vai ver a imagem q esta em uma pasta dentro do nosso propio cuputador

para adicionar uma imagem adicone a tag (<img src="origem do arquivo" alt="alternativa" ) ou somente digite img e aperte enter e o vs code ira abrir a tag automaticamente.
para adicionar a imagem sem precisar especificar seu caminho adicione a imagem junto ao arquivo no vs code, adicione alternativa para que a ferrament de pesquisa saiba do que se trata a imagem (nao obrigatorio)

carregando imagens externas

tambem podemos carregar imagens de um site ou link
para carregar a imagem de um link basta copiar o endereço da imagem e colar sobre a tag (<img src="endereço da imagem" alt="alternativa") exatamente igual para carregr uma imagem interna porem com o endereço (link) desta imagem

ajustando a imagem conforme o tamanho da janela

faça o teste na imagem a cima e vera que podemos fazer com que a imagem se ajuste confore o tamanho que a tela estiver aberta isso facilitara a visualização de sua pagina para usuarios de outros dispositivos como smartfones.

separe a mesma imagem com tamanhos dferentes

voce tera que ter 3 ou mais imagem de tamanhos diferentes recomendado que seja uma de 1200x uma de 700 e outra de 300x lembrando que esses tamanhos não são uma regra.

faça a troca de imagem

para ajustar sua imagem voce tera que fazer com que o site troque a imagem atual por uma imagem menor para fazer isso basta usar a tag (picture) dento da tag adicione outra tag (source) na tag source dentro da tag source adicione a função media="" dentro das aspas da função voce ira digitar entre parenteses (max-width:"o tamanho desejado"px) se a tela do usuario estiver menor que o tamanho especificado nesta ultima função o site automaticamente trocara para a imagem menor que vou te ensinar agora como adicionala

adicionando imagem a ser trocada

ainda dentro da nossa tag source depos da funçao media="" adicione a funçao srcser="o nome da imagem" apos isso adicione a funçao type="image/"o tipo da imagem""
para adionar outra troca de imagem menor basta executar os mesmo codigo a cima desse que acabamos de criar lembrando que sempre sera da menor para a maior e a imagem com tamanho padrao sempre devera ser a maior e sempre sera a ultima abaixo voce vera o codigo da imagem acima com uma imagem padrao e duas trocas para menores...

adicionando icone na pagina

para adionar um icone em sua pagina basta voce ter uma imagem em .ico, va ate a parte de "cabeça" do codigo de sua pagina e adicionar o nome da imagem sobre a tag (<link rel="shortcut icon" href="nome da imagem" type="image/x-icon">)
ou somente digite (link) e o vs code ira abrir uma lista com opçoes e confirme a opçao (link:favicon)

adicionando videos

adionando videos internos

para adicionar videos internos é facil, basta usar a tag (video) dentro da tag voce ira colocar o comando (src="nome do video") logo em seguida voce colocara o comando (controls) e para colocar a o tamanho dentro da tag voce usara o comando (width="o tamanho desejado") e para definir o tamanho voce ira usar o comando (height="a largura desejada") veja só como ira ficar

<video scr="nome do video" controls width="tamanho desejado" height="largura desejada">

adicionando audios

podemos tambem adicionar audio ao nosso site e até mesmo fazer com que ele seja executado automaticamente (opção automatica não disponivel para o google).
para adionar audio basta usar a tag (audio) dentro da parte <audio> voce ira digitar preload="escolha uma opção" logo após digite controls
ficando assim (<audio preload="escolha uma opção " controls>) logo apos isso voce ira usar a tag (source src="o nome do audio" type="audio/tipo do arquivo") ficando asssim: 

        
        
<audio preload="escolha uma opção" controls>
<source src="nome do audio" type="audio/formato do audio">
</audio>
