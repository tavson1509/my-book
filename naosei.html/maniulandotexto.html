principais formataçoes

negrito/destaque

nessa frase temos um termo em negrito usando usando a tag (<b>) sendo (<b>"digite o texto"</b>)

nessa frase temos um termo em destaque usando a tag (<strong>) sendo (<strong>"digite o texto"</strong>)

italico/enfase


para escrever em italico basta adicionar o texto desejado entre a tag (<i>) sendo (<i>"digite o texto"</i>) ou basta voce selecionar o texto desejado e o envelopar apertando as teclas ctrl + shift + p e digite e selecione a opçao (emmt: envelope com abreviatura)

texto marcado

podemos criar tambem um texto marcado usando a tag mark sendo
(<mark>"digite seu texto"</mark>)
ou apenas o envelope usando a opcao mark
caso voce queira trocar a cor da marcaçao voce precisara usar uma tag em css que seria style="background-color:"a cor desejada;"" sendo assim
(<mark style="background-color:"a cor desejada;">"dgite o texto""</mark>)
caso voce queira que todas as marcaçoes padrao de seu progeto seja da mesma cor voce tera que colocar na parte de "cabeça" do seu projeto a tag <style> {background-color"escolha a cor";}</style>

texto deletado/texto incluido

podemos marcar um texto " deletado " que pode ser lido mas nao considerad, basta usar a tag (del) e ecrever o texto desejado sobre ela

para dar enfase e indicar q um texto foi incluido depois basta usar a tag (ins) e escrever o texto desejado sobre ela

outras fontes

fonte mono espaçadas

caso voce deseje adiocionar algo que nao pode ser confundido tipo um email tavison072010@gmail.com e voce quer da espaço para as letras use o a tag (code) e adicione o texto desejado sobre ela tavison072010@gmail.com

identaçao

caso voce queira escrever algo identado tipo um codigo de progamaçao voce pode usar a tag (pre) exemplo

num = int(input("digite um numero")) if num != 1: print("o numero {}invalido tente novamente".format(num)) else: print("ok vamos prosseguir") print("fim do programa")

como podemos ver nao tem identaçao assim confindindo quem esta lendo o codigo, usando a tag (pre) ficaria 

        num = int(input("digite um numero"))
        if num != 1:
            print("o numero {}invalido tente novamente".format(num))
        else:
            print("ok vamos prosseguir")
        print("fim do programa")

adicionando links

adicionando links externos

para adionar um link exemplo repositorio do guanabara no git hub
basta usar a tag (a) sendo assim
(<a href="escreva o link" rel= "external">"digite o texto que voce deseja por sobre o link"</a>)
mas atençao dessa forma o o site do link digitado ira abrir por cima do da janela atual para abrir o link em uma nova janela use o comando (target="_blank") depois de ter digitado o link e antes do comando (rel) ficando assim:
(<a href="digite o link" target="_blank" rel="external>digite o texto que voce deseja por sobre o link</a>)
