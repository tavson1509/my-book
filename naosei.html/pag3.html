    • manipulando imagens

    ola aqui vamos aprender a colocar imagens em nossas "box" ajustar tamanho, posição e ate mesmo as colocar de fundo, mas lembre-se sempre use imagens de seu direito e que tenha tons para encaixar com o conteudo do seu site

    adicionando imagem ao fundo

    para adionar uma imagem de fundo no seu site basta no estilo do seu "body" adicionar a tag:
    background-image: url=(o caminho e o nome da imagem);
    porem a imagem ira ficar repetida aparecendo varias imagens repetidas pela sua tela para que isso nao aconteça use a tag:
    background-repeat: no-repeat;
    voce vera que a imagem nao esta repetindo mais, porem a imagem acaba quando o site atinge seu tamanho, temos duas soluçoes para isso, uma usando a tag
    background-position: 100% 100%;
    porem voce vera que a imagem ira se esticando e contraindo conforme a janela for reduzindo ou aumentando de tamanho fazendo a imagem ficar totalmente distorcida o que não é muito bom dependendo da imagem mas temos também a tag:
    background-size:cover;
    desse jeito a imagem nao sera distorcida porem se a janela for menor que a propia a imagem sera cortada priorzando o ponto de destaque dela. nesse caso a imagem ira rolar junto com a aba, para que ela fique fixa use a tag:
    background-attachment: fixed;

    adicinando imagem nas divs

    aqui vamos aprender a colocar imagens no fundo de nossas divs para isso basta nas configurações de stilos da div adicionar a tag
    background-image: url(aqui digite o caminho e nome da imagem)
    simples assim use a tag:
    background-size:cover;
    para a imagem ficar inteira em sua div e use a tag:
    background-attachment:fixed;
    para que a imagem fique fixa enquanto a aba é movimentada

    adicionando iframe

    o que é um iframe ?

    aqui vamos apreder a usar o iframe, o iframe pode ser usado para ter em seu conteudo um site que você tenha ou um site de terceiros dentro de uma aba em sua pagina, é bastante simples e facil usar um iframe e no dia a dia do desenvolvedor uma tag importante e simples, clique nos links a baixo e entenda mais sobre o iframe...

    como usar iframe ?

    para usar um iframe é facil basta usar a tag:
    <iframe src"digite o nome do site">
    pronto simples assim, mas lembre-se voce ainda pode fazer ajuste de tamanho e estilo dentro da sua css

    como ativar por link ?

    é facil basta usar a tag a e adicionar o link do site desejado a ela mas dentro do tag <iframe> voce ira adicionar o comando name="qualquer nome" pronto quando voce usar a tag a voce ira adicionar dentro dela o comando target="o nome de sua iframe" ficando assim:
    <iframe src="" name="qualquer nome">
    <a href="link do site" target="o nome do iframe">
    viu como é facil e simples ?

    adicionando formularios

    o que é um formulario ?

    ola chegamos aqui e agora vamos usar tags q possibilitam nossos usuarios de interagir com nosso site, de uma maneira bem simples também, por mais que HTML e CSS tenham bastantes tags muitas dela inclusive as principais sao faceis de serem usadas, a nossa tag <form> também nao sera dificil. veja um exemplo a baixo e nao se preocupe. pois seus dados nao serão enviados para nenhum local...

    isso que é um formulario, um campo aonde o usuario deixa seus dados ou exigencias agora vamos aprender como faz!!!

    como fazer formulario ?

    vamos começar usando a tag <form> entre ela voce tera varias opções de tags input para usar, logo a baixo te mostrarei os pincipais input e para que eles servem.

    <input type="text" name="de um nome" id="de um nome">
    essa tag é para o usuario digitar qualquer texto

    <input type="email" name="de um nome" id="de um nome">
    essa tag é para o usuario digitar o email

    <input type="tel" name="de um nome" id="de um nome">
    essa tag é para o usuario digitar o numero de telefone

    <input type="date" name="de um nome" id="de um nome">
    essa tag é para o usuario escolher uma data

    <input type="reset" value="escreva dentro do botão">
    essa tag é para criar um botão para se o usuario desejar limpar o que ele escreveu

    <input type="submit" value="escreva dentro do botão">
    essa tag é para o usuario confirmar e enviar o formulario
