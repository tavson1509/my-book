usando cores

cor de fundo do site

use a cor desejada no fundo de seu site use cor que va fazer sentido com as cores usadas em seu cunteudo e com a lgo de seu site ou marca, para adionar uma cor ao fundo de seu site dentro de <body> adiocione o comando style="background-color:escolha uma cor" ficando assim
<body style="background-color:escolha uma cor">

adionando cores nas letras e fundos

adionar cores nas letras é algo bem usado e para fazer é bem simples tambem se você deseja adionar cores nas letra de um titulo basta dentro da tag <h1> adionar o comando style="color:escolha uma cor;" se voce desejar adicionar uma cor de fundo tambem basta digitar depois de style="escolha uma cor; voce digitar backgrund-color:escolha uma cor;" ficando assim.
<h1 style="color:escolha uma cor; background-color;escolha uma cor;">
o mesmo vale para a tag <p> ou para outras variantes de titulos como h2, h3...
